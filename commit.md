# ***git commit***

Команда ` git commit` – это команда Git для записи индексированных изменений в репозиторий. Используйте эту метку для обозначения всех вопросов, связанных с созданием, редактированием и внутренней структурой коммитов в Git. Общие сведения. git commit - это команда для записи индексированных изменений в репозиторий Git. Прежде чем создавать очередной коммит, необходимо проиндексировать файлы в рабочей области с помощью команды git-add.
